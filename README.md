# Post-picture

## Overview

個人開発アプリケーションです。
canvasを利用したペイントのようなものを実装、お絵かきを投稿できるアプリケーションです。

## Description

＊使用言語
Ruby/Ruby on Rails/JavaScript/HTML(canvas)/SASS

タブレット端末、スマートフォンでの操作を想定し、全てのページをレスポンシブ対応
マウス操作、タッチ操作ともに描画が可能
